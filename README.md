# MixReflect
 
MixReflect is a two-sided music feedback marketplace where artists submit tracks for structured reviews from genre-matched reviewers.

## Documentation

Start here:

- `docs/README.md`

## Local development

```bash
npm install
cp .env.example .env
npm run db:push
npm run db:seed
npm run dev
```

Open `http://localhost:3000`.
